# Anomaly Detection System

A machine learning pipeline for detecting abnormal behaviour in renewable energy systems.

---

## Overview

This module identifies unusual patterns in system data that may indicate faults or inefficiencies. Early detection enables proactive intervention and reduces system downtime.

---

## Problem Statement

Energy systems generate large volumes of operational data. Detecting anomalies is challenging due to:

- Lack of labelled anomaly data  
- Complex and dynamic system behaviour  

This module applies unsupervised learning techniques to address these challenges.

---

## Data

- Simulated sensor and operational data  
- Time-series system behaviour patterns  

---

## Methodology

### Data Processing
- Cleaning and normalisation  
- Feature scaling  

---

### Models Used

- **Isolation Forest** → identifies anomalies based on data isolation  
- **Statistical Thresholding** → flags extreme deviations  

---

## Results

- Successfully detected abnormal system behaviour across simulated scenarios  
- Enabled early identification of potential faults  

---

## Visualisations

- Anomaly detection plots

- Distribution analysis  

---

## Key Takeaways

- Unsupervised models are effective for anomaly detection  
- Thresholding enhances detection precision  
- Early anomaly detection supports proactive maintenance
