# Predictive Maintenance (RUL Estimation)

A machine learning model to estimate the Remaining Useful Life (RUL) of assets for proactive maintenance planning.

---

## Overview

This module predicts how long equipment will continue to operate before failure. It enables data-driven maintenance strategies that reduce downtime and operational costs.

---

## Problem Statement

Traditional maintenance approaches are:

- Reactive (after failure)  
- Preventive (fixed schedules)  

Both approaches are inefficient. This module introduces predictive maintenance using machine learning.

---

## Data

- Simulated equipment and operational data  
- Time-dependent degradation patterns  

---

## Methodology

### Feature Engineering
- Time-based degradation indicators  
- Rolling averages and trend features  

---

### Model Used

- **Gradient Boosting Regression** → captures complex relationships in degradation patterns  

---

## Results

- Estimated Remaining Useful Life across simulated operational scenarios  
- Demonstrated strong predictive capability for maintenance planning  

---

## Visualisations

- Predicted vs Actual RUL plots  
- Feature importance analysis  
- Degradation trend visualisations  

---

## Key Takeaways

- Predictive maintenance reduces downtime and costs  
- Feature engineering is critical for RUL modelling  
- Gradient boosting models perform well on structured time-series data
